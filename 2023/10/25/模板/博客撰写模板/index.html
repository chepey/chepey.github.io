博客撰写模板

一、博客上传

创建一个新博客

$ hexo new "My New Post"

More info: Writing

清除

$ hexo clean

预览页面

$ hexo server

More info: Server

生成页面

$ hexo generate

More info: Generating

部署到github服务器

$ hexo deploy

二、删除文章

删除文章：
1、删除文件夹source/_posts下目标文章markdown文件
2、删除.deploy_git文件夹
3、执行hexo clean后，再执行hexo g，hexo d 即可

| ------------ | -------------------------------------------- |
| S-T          | 返回一个新集合，包括在S内但不包含在T内的元素 |
| S&T          | 返回一个新集合，包括同时在S和T内的元素       |
| S^T          | 返回一个新集合，包括S和T中非共同元素         |
