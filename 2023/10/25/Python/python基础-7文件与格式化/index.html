文件操作

一、文件打开和关闭

open()

<变量名>=open(<文件路径及文件名>,<打开模式>)

  • ‘r’ 只读模式，如果文件不存在，返回FileNotFoundError
  • ‘w’ 覆盖写模式，文件不存在则创建，存在则完全覆盖
  • ‘x’ 创建写模式，文件不存在则创建，存在则返回FileExistError
  • ‘a’ 追加写模式，文件不存在则创建，存在再源文件内容最后追加内容
  • ‘b’ 二进制文件模式
  • ‘t’ 文本文件模式，默认值
  • ‘+’ 与r/w/x/a一同使用再原功能基础上增加同时读写功能
关闭

<变量名>.close()

二、文件的读写

  • f.read(size=-1) 从文件中读入整个文件内容。参数可选，如果给出读入前size长度的字符串或字节流
  • f.readline(size=-1) 从文件中读入一行内容，参数可选，如果给出，读入该行前size长度的字符串或字节流
  • f.readlines(hint=-1) 从文件读入所有行，以行为元素形成一个列表。参数可选，如果给出，读入hint行
  • f.seek(offset) 改变当前文件操作指针位置：offset：0为文件头2为文件尾

如果文件以文本方式打开，则读入字符串，以二进制方式打开则读入字节流

  • 可以通过遍历循环逐行遍历文件

    f=open("bar.txt","r")
for line in f
    print(line)
f.close()

  • f.write(s) 向文件写入一个字符串或字节流

  • f.writelines(lines) 将一个元素为字符串的列表整体写入文件

数据组织的维度

一、一维数据

（1）列表形式表示
  • is=[‘1’,’2’]
（2）一维数据的存储

  • 用空格，逗号，换行分隔元素，其他特殊符号

  • 逗号分隔的存储格式叫做CSV格式，存储文件一般采用*.csv

    is=['beijin','shanghai','tianjing','chongqing']
f=open('chengshi.csv','w')
f.write(",".join(is)+'\n')
f.close()
（3）一维数据的处理
f=open('chengshi.csv','r')
is=f.read().strip('\n').split(',')	#去掉换行符，以逗号分割
f.close()

二、二维数据

（1）二维列表形式表示
  • is=[[‘’,’’],[‘’,’’],[‘’,’’]]
（2）二维数据的存储

  • 用CSV存储，每一行是一个一维数据

    is=[['beijin'],['shanghai'],['tianjing'],['chongqing']
f=open('chengshi.csv','w')
for line in is
    f.write(",".join(line)+'\n')
f.close()
（3）二维数据的处理
f=open('chengshi.csv','r')
is=[]
for line in f
    is.append(f.read().strip('\n').split(','))	#去掉换行符，以逗号分割
f.close()
