3.3字符串类型&格式化

print('''这是'多行字符串'的第一行
这是'多行字符串'的第二行''')#此处换行符表示输出的换行
  • \表示转义字符
  • \也可续行

3.3.1字符串的索引

  • <字符串或字符串变量>[序号]

3.3.2字符串的切片

  • <字符串或字符串变量>[N:M]
  • 从N到M，不包含M

3.3.3format()方法的使用

  • <模板字符串>.format(<逗号分隔的参数>)
"{}曰：学而时习之不亦{}".format("孔子","说乎")
"{1}曰：学而时习之不亦{0}".format("说乎","孔子")
  • 如果希望在模板字符串中输出大括号，使用\lbrace\rbrace表示

3.3.4format()方法的格式控制

  • {<参数序号>:<格式控制标记>}

  • 六个格式控制可以分为两组<填充><对齐><宽度>
s="sssss"
"{:*^25}.format(s)#居中对齐，填充*号，宽度25个字符
  • <，><.精度><类型>
s=1234.56789
"{:*^25,.3f}.format(s)#居中对齐，填充*号，宽度25个字符,数中间用，分割，输出三位小数的浮点数

b：输出整数的二进制方式；

c：输出整数对应的Unicode字符；

d：输出整数的十进制方式；

o：输出整数的八进制方式；

x：输出整数的小写十六进制方式；

X：输出整数的大写十六进制方式；

e：输出浮点数对应的e的指数形式；

E：输出浮点数对应的E的指数形式；

f：输出浮点数标准格式；

%：输出浮点数的百分比形式；

3.4字符串类型的操作

3.4.1字符串操作符

  • x+y，连接两个字符串
  • x*n，复制n次字符串
  • x in s，如果x是s的子串，返回True

3.4.2字符串处理函数

len(x)#x的长度
str(x)#任意类型x所对应字符串的形式
chr(x)#Unicode编码x对应的单字符
ord(x)#x对应的Unicode编码
hex(x)#整数x对应的十六进制的小写形式字符串
oct(x)#整数x对应的八进制的小写形式字符串

3.4.3字符串处理方法

str.lower()#全部字符串小写
str.upper()#全部字符串大写
str.split(sep=None)#返回一个列表，有str根据sep被分割的部分构成，省略sep默认以空格分隔
str.count(sub)#返回sub子串出现的次数
str.replace(old,new)#替换old
str.center(width,fillchar)#字符串居中函数，fillchar参数可选
str.strip(chars)#从str中去掉在其左侧和右侧chars中列出的字符
str.join(iter)#将iter变量的每一个元素后增加一个str字符串
