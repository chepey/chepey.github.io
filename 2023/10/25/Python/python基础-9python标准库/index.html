python标准库

turtle

作用：基本图形绘制

9.1引用方式：

1.import，则对turtle库中的函数调用采用turtle.<函数名>()形式

import turtle
turtle.circle(200)

2.from turtle import *，则对turtle库中函数调用直接采用<函数名>()形式，不再使用turtle.作为前导

from turtle import *
circle()

3.import turtle as t，则对turtle库中函数调用采用更简洁的t.<函数名>()形式，保留字as的作用是将turtle库给予别名t

import turtle as p
p.circle(200)

9.2.1窗体函数
turtle.setup(width,height,startx,starty)
9.2.2画笔状态函数
函数描述
pendown()放下画笔
penup()提起画笔，与pendown()配对使用
pensize(width)设置画笔线条的粗细为指定大小
pencolor(colorstring)设置画笔的颜色
color(colorstring1,colorstring2)设置画笔和填充颜色
begin_fill()填充图形前，调用该方法
end_fill()填充图形结束
filling()返回填充的状态，True为填充，False为未填充
clear()清空当前窗口，但不改变当前画笔位置
reset()清空当前窗口，并且重置位置等状态为默认值
screensize(width,height,bgcolor)设置画布窗口的宽度、高度和背景颜色
hideturtle()影藏画笔的turtle形状
showturtle()显示画笔的turtle形状
isvisible()如果turtle可见，返回Ture
write(str,font=None)输出font字体的字符串

具体参数查web/书P157

9.2.3画笔运动函数
函数描述
forward(distance)别名fd(distance)沿当前方向前进指定距离
backward(distance)别名bk(distance)沿当前方向后退指定距离
right(angle)向右旋转angle角度
left(angle)向左旋转angle角度
goto(x,y)移动到绝对坐标(x,y)处
setx(x)修改画笔的横坐标到x，纵坐标不变
sety(y)修改画笔的纵坐标到y，横坐标不变
setheading(angle)设置当前朝向为angle角度
home()设置当前画笔位置为原点，朝向东
circle(radius,e)绘制一个指定半径r和角度e的圆或弧形
dot(r,color)绘制一个指定半径r和颜色color的圆点
undo()撤销画笔最后一步动作
speed()设置画笔的绘制速度

random库

9.3random库与随机数运用

函数描述
seed(a=None)初始化随机数种子，默认值为当前系统时间
random()生成一个[0.0,1.0)之间的随机小数
randint(a,b)生成一个[a,b]之间的整数
getrandbits(k)生成一个k比特长度的随机整数
randrange(start,stop,step)生成一个[start,stop)之间以step为步数的随机整数
uniform(a,b)生成一个[a,b]之间的随机小数
choice(seq)从序列类型（例如列表）中随机返回一个元素
shuffle(seq)将序列类型中元素随机排列，返回打乱后的序列
sample(pop,k)从pop类型中随机选k个元素，以列表类型返回

time库

9.4time库概述

用来分析程序性能，也可以让程序暂停运行

9.4.1时间处理
（1）time.time()
  • 获取当前时间戳
（2）time.gmtime(secs)
  • 获取当前时间戳对应的struct_time对象。
（3）time.localtime(secs)
  • 获取当前时间戳对应的本地时间的struct_time对象（类似北京时间）
（4）time.ctime(secs)
  • 获取当前时间戳对应的易读字符串表示，内容会调用time.localtime()函数以输出当地时间
9.4.2时间格式化
（1）time.mktime()

  • 使用time.mktime(t)将struct_time对象t转换为时间戳，注意t代表当地时间。struct_time对象的元素构成见表

    下标属性值
    0tm_year年份，整数
    1tm_mon月份[1,12]
    2tm_mday日期[1,31]
    3tm_hour小时[0,23]
    4tm_min分钟[0,59]
    5tm_sec秒[0,61]
    6tm_wday星期[0,6]（0表示星期一）
    7tm_yday该年第几天[1,366]
    8tm_isdst是否为夏令时，0否，1是，-1未知
    （2）time.strftime()

    • 时间格式化最有效的方法，利用一个格式字符串，对时间格式进行表示

      time.strftime("%Y-%m-%d %H:%M:%S",lctime)
      格式化字符串日期/时间值范围
      %Y年份0001~9999
      %m月份01~12
      %B月名January~December
      %b月名缩写Jan~Dec
      %d日期01~31
      %A星期Monday~Sunday
      %a星期缩写Mon~Sun
      %H小时（24h制）00~23
      %I小时（12h制）01~12
      %p上/下午AM,PM
      %M分钟00~59
      %S秒00~59
      （3）time.strptime()
      • 与strftime()相反，用于提取字符串时间生成struct_time对象啊你个
