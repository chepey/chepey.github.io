调用逻辑

oled的显示依赖于屏幕中的驱动芯片，XDrive中选用的是SSD1306芯片，这个芯片采用SPI协议驱动，因此这个模块里也包含了STM32芯片的SPI的基本使用方法。

基于的头文件

//Oneself
#include "ssd1306.h"

//Font
#include "oled_font.h"

宏定义

工程配置

这里包含了整个Oled的行列像素个数

#define Quantity_Row		128		//行
#define Quantity_Column		32		//列

像素空间（实例）

分配一个外部变量构成一个矩阵，用以存放当前需要显示的像素信息。

这个矩阵空间大小为row*col，由于存放在unsigned int （8位）的变量中，因此这个数组应该/8（1个bit代表一个像素亮度）。

这个gram空间哎ssd1306.c中定义，并在ssd1306.h中extern，供外部访问，所有引用到这个头文件的源代码都可以访问本gram。

extern uint8_t	ssd1306_gram[(Quantity_Row) * (Quantity_Column/8)];	//GRAM空间

GPIO输出

通过寄存器的方法，读取按键的值。

其中BUTTON_OK_GPIO_Port、BUTTON_OK_Pin等宏定义在XDrive_REIN_Basic_H1_0.h文件的BUTTON选项下。

#define SSD1306_SPI_CS_H()		(OLED_SPI_CS_GPIO_Port  -> BSRR = OLED_SPI_CS_Pin )
#define SSD1306_SPI_CS_L()		(OLED_SPI_CS_GPIO_Port  -> BRR  = OLED_SPI_CS_Pin )
#define SSD1306_SPI_RES_H()		(OLED_SPI_RES_GPIO_Port -> BSRR = OLED_SPI_RES_Pin)
#define SSD1306_SPI_RES_L()		(OLED_SPI_RES_GPIO_Port -> BRR  = OLED_SPI_RES_Pin)
#define SSD1306_SPI_DC_H()		(OLED_SPI_DC_GPIO_Port  -> BSRR = OLED_SPI_DC_Pin )
#define SSD1306_SPI_DC_L()		(OLED_SPI_DC_GPIO_Port  -> BRR  = OLED_SPI_DC_Pin )

类型定义

按钮状态类型

字体大小为枚举变量，它的定义用于确定当前字符字体的大小。

typedef enum{
	CharFont_0806 = 0x01,		//0806字符
	CharFont_1206,					//1206字符
	CharFont_1608,					//1608字符
	CharFont_2412,					//1608字符
}CharFont;

接口函数

读写操作

DC引脚先高后低，先低后高来区分是写数据还是写指令

void SSD1306_Write_Data(uint8_t _data)

  • 功能：写数据

  • 参数：

    • _data：输出的数据

  • 返回：

  • 实现：

    • 拉低片选引脚
    • DC引脚拉高
    • 通过HAL库函数HAL_SPI_Transmit()向SPI实例OLED_SPI_Get_HSPI输出_data数据
    • DC引脚拉低
    • 拉高片选引脚
void SSD1306_Write_Comm(uint8_t _comm)

  • 功能：写指令

  • 参数：

    • _comm：输出的指令

  • 返回：

  • 实现：

    • 拉低片选引脚
    • DC引脚拉低
    • 通过HAL库函数HAL_SPI_Transmit()向SPI实例OLED_SPI_Get_HSPI输出_comm数据
    • DC引脚拉高
    • 拉高片选引脚

基本数据操作

void SSD1306_Refresh(uint8_t *_gram)
  • 功能：更新显示
  • 参数：
    • _gram：gram的地址
  • 返回：
  • 实现：
    • 遍历每一列
      • 输出指令0xB0 + 列序号
      • 输出指令0x00
      • 输出指令0x10
      • 遍历每一行
        • 按字节输出gram中x行 + 列号*Quantity_Row位置处的数据

接口函数

void SSD1306_Init(void)
  • 功能：SSD1306初始化
  • 参数：
  • 返回：
  • 实现：
    • 初始化GPIO
    • 初始化SPI
    • 拉低Oled复位引脚，延时后拉高
    • 初始化序列
      • 输出指令0xAE，关闭显示
      • 输出指令0x00，设置最低列的地址
      • 输出指令0x10，设置最高列地址
      • 输出指令0x00，设置显示开始行
      • 输出指令0xB0，设置页地址
      • 输出指令0x81，合同控制
      • 输出指令0xff，128
      • 输出指令0xA1，设置段重映射
      • 输出指令0xA6，方向：正/反
      • 输出指令0xA8，可设定程序的比例
      • 输出指令0x1F，占空比1/32
      • 输出指令0xC8，Com扫描方向
      • 输出指令0xD3，设置显示偏移
      • 输出指令0x00
      • 输出指令0xD5，设置osc划分
      • 输出指令0x80
      • 输出指令0xD9，设置预充电周期
      • 输出指令0x1f，
      • 输出指令0xDA，设置 COM 引脚
      • 输出指令0x00，
      • 输出指令0xdb，设置 vcomh
      • 输出指令0x40，
      • 输出指令0x8d，设置充电泵使能
      • 输出指令0x14，
      • 输出指令0xAF，开启显示
    • 方向翻转
