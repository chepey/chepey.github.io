调用逻辑

MT6816有三种读角度的方法，分别为PWM、SPI、ABZ。因此在这个模块中，通过定义在头文件中的宏定义变量MT6816_Mode来确定初始化的函数模块。

因此，在本文档中，将分别从三种读角度方法的方向总结MT6816的使用方法。

基于的头文件

//Oneself
#include "mt6816.h"

//Config
#include "stockpile_f103cb.h"

//Control
#include "control_config.h"

宏定义

模式配置

用于配置支持的MT6816工作模式，文档内不同的模式被通过选择性编译。

//MT6816工作模式定义
#define MT6816_Mode_PWM		(0x01)	//MT6816工作在PWM模式		
//(支持该模式的硬件版本：XDrive_REIN_Basic_H1_0)
#define MT6816_Mode_ABZ		(0x02)	//MT6816工作在ABZ模式		
//(支持该模式的硬件版本：XDrive_REIN_Basic_H1_0)
#define MT6816_Mode_SPI		(0x03)	//MT6816工作在SPI模式		
//(支持该模式的硬件版本：XDrive_REIN_Basic_H1_0 / XDrive_REIN_Basic_H1_1)

//MT6816工作模式配置
#define MT6816_Mode		MT6816_Mode_SPI

矫正数组读取

uint16_t *Read_QuickCali_DATA = (uint16_t*)STOCKPILE_APP_CALI_ADDR;

GPIO输出

#define MT6816_SPI_CS_H()		(MT6816_SPI_CS_GPIO_Port -> BSRR = MT6816_SPI_CS_Pin)
#define MT6816_SPI_CS_L()		(MT6816_SPI_CS_GPIO_Port -> BRR  = MT6816_SPI_CS_Pin)

}MT6816_Typedef;
extern MT6816_Typedef	mt6816;

PWM

宏定义

工程配置

用于存放PWM采集到的信息

MT6816_PWM_Signal_Typedef mt6816_pwm;

类型定义

typedef struct{
	//采集数据(PWM)
	uint16_t	h_width;		//PWM高宽度
	uint16_t	period;			//PWM周期
	uint8_t		count_rising;	//PWM上升沿计数器镜像
	uint8_t		count_falling;	//PWM下降沿计数器镜像
	uint8_t		count_update;	//PWM更新计数器
	bool		whole_h_flag;	//PWM全高标志
	bool		whole_l_flag;	//PWM全低标志
	bool		ready_once;		//PWM就绪标志(完成一次PWM采集后置位)
	bool		ready_again;	//PWM就绪标志(完成二次PWM采集后置位)
	uint16_t	valid_width;	//PWM有效宽度
	//输出数据(PWM)
	uint16_t	angle;						
}MT6816_PWM_Signal_Typedef;

  • 功能：MT6816_PWM采集中断回调
  • 参数：
  • 返回：
  • 实现：
    • 上升沿中断处理：
      • 清除中断标志位
      • 读取PWM周期

    • 下降沿中断处理：
      • 清除中断标志位
      • 读取PWM高电平宽度
      • 读取PWM下降沿计数值（即更新计数器的值）
    • 更新事件中断处理
      • 清除中断标志位
      • 单次PWM全高全低检测（全高时，更新计数器自加）
      • 可靠的PWM有效性确认
      • 令输出有效宽度=高电平宽度

void RINE_MT6816_SPI_Get_AngleData(void)

  • 功能：MT6816_SPI采集获取角度数据

  • 参数：

  • 返回：

  • 实现：

    • 生成tx数据

      • data_t[0] = (0x80 | 0x03) << 8;代表高8位
      • data_t[1] = (0x80 | 0x04) << 8;代表低8位

    • 读取SPI数据

      • 发射高八位地址并读数据
      • 发射低八位地址并读数据
      • 合成数据

    • 奇偶校验

      • 遍历合成后数据每一位，判断奇偶

    • 当为偶数时：

      • 输出角度

      • 检测磁铁是否有效
